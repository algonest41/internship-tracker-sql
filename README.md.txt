# Internship Tracker SQL Project

This is a beginner SQL project made using MySQL.

The project helps manage:
- Students
- Companies
- Internship applications
- Interview records

## Technologies Used
- SQL
- MySQL Workbench

## Tables Used
- companies
- students
- applications
- skills
- interviews

- CREATE TABLE
- PRIMARY KEY
- FOREIGN KEY
- INSERT
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- UPDATE
- DELETE
- ALTER TABLE

## Files
- schema.sql
- insert_data.sql
- queries.sql

## Purpose
This project was created to practice basic SQL concepts and database operations.
